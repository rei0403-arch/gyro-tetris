Gyro Tetris

폰을 기울여 움직이고, 폰을 돌려 회전하는 안드로이드 테트리스. 방 코드로 친구랑 대전하고, 2줄을 지울 때마다 상대에게 잉여블록을 보낼 수 있어.

APK 다운로드
https://rei0403-arch.github.io/gyro-tetris/
현재 빌드
Firebase 익명 로그인, 전체 랭킹, 방 코드 멀티, 콤보 점수, 자연 회전, 낙하 미리보기, 감도 설정, 일시정지/리트라이, 꾹 눌러 입력 잠금 포함.

친구 설치 방법

보내는 사람
  1. 이 페이지 링크를 친구에게 보내기
  2. 친구가 APK 다운로드를 누르게 하기
  3. 설치가 끝나면 둘 다 MULTI로 같은 방 코드에 들어가기
받는 사람
  1. 안드로이드 폰에서 링크 열기
  2. APK 다운로드 후 열기
  3. 설치 차단이 뜨면 브라우저의 앱 설치 허용 켜기
  4. 뒤로 가서 설치 누르기

조작

이동폰을 좌우로 기울이기. 많이 기울일수록 더 빠르게 움직여.
회전폰을 자연스럽게 돌리기. 블록이 놓일 때는 가장 가까운 직각으로 고정돼.
빠른 낙하음량 위/아래 버튼을 동시에 누르기.
설정SET에서 이동/회전 감도를 미리보기 보면서 조절.
입력 잠금화면을 꾹 누르는 동안 기울기/회전 입력이 멈춰.
멀티MULTI로 방을 만들거나 친구 방 코드에 입장. 2줄마다 잉여블록 1줄 전송.

랭킹

앱은 Firebase Anonymous Auth와 Cloud Firestore를 사용해. 앱에서 닉네임을 입력하고 플레이하면 최고 점수가 전체 랭킹에 올라갈 수 있어.

Firebase enabled

방 코드 대전

멀티는 Firebase Realtime Database를 사용해. 한 명이 방을 만들고, 다른 한 명이 화면에 나온 코드를 입력하면 플레이 중 잉여블록이 동기화돼.

Room codes enabled
