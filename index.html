자이로 테트리스

폰을 기울여 움직이고, 폰을 돌려 회전하는 안드로이드 테트리스. 싱글플레이와 숫자 6자리 방 코드 멀티플레이를 지원하고, 멀티에서는 승리/패배가 자동으로 표시돼.

APK 다운로드
https://rei0403-arch.github.io/gyro-tetris/
현재 빌드
시작화면, 싱글/멀티 선택, 숫자 방 코드, 상대 입장 전 대기, 승리/패배 화면, 멀티 다시하기 비활성화, 창이 열려 있을 때 일시정지, 전체 랭킹 포함.

친구 설치 방법

보내는 사람
  1. 이 페이지 링크를 친구에게 보내기
  2. 친구가 APK 다운로드를 누르게 하기
  3. 설치가 끝나면 한 명은 멀티플레이에서 방 만들기
  4. 다른 한 명은 숫자 6자리 방 코드로 입장
받는 사람
  1. 안드로이드 폰에서 링크 열기
  2. APK 다운로드 후 열기
  3. 설치 차단이 뜨면 브라우저의 앱 설치 허용 켜기
  4. 뒤로 가서 설치 누르기

게임 시작

싱글플레이혼자 바로 시작해. 앱을 켜도 플레이 버튼을 누르기 전까지는 게임이 시작되지 않아.
멀티플레이방을 만들거나 친구의 숫자 6자리 코드를 입력해서 들어가.
대기멀티는 상대가 들어오기 전까지 블록이 떨어지지 않고 멈춰 있어.

멀티 승패

자동 종료한 명이 게임오버 되면 상대 화면도 라운드가 끝나고 승리 또는 패배가 표시돼.
다시하기 제한멀티 중에는 다시 버튼이 비활성화돼. 새 판은 방을 다시 만들거나 나간 뒤 시작하면 돼.
창 열림 정지설정, 랭킹, 닉네임, 멀티 메뉴 같은 창이 떠 있으면 블록 낙하와 입력이 잠시 멈춰.

조작

이동폰을 좌우로 기울이기. 많이 기울일수록 더 빠르게 움직여.
회전폰을 자연스럽게 돌리기. 블록이 놓일 때는 가장 가까운 직각으로 고정돼.
빠른 낙하음량 위/아래 버튼을 동시에 누르기.
설정설정에서 이동/회전 감도를 미리보기 보면서 조절.
입력 잠금화면을 꾹 누르는 동안 기울기/회전 입력이 멈춰.
공격멀티에서 2줄을 지울 때마다 상대에게 잉여블록 1줄을 보내.

랭킹

앱은 Firebase 익명 로그인과 Cloud Firestore를 사용해. 닉네임을 입력하고 플레이하면 최고 점수가 전체 랭킹에 올라갈 수 있어.

랭킹 켜짐

방 코드 대전

멀티는 Firebase Realtime Database를 사용해. 한 명이 방을 만들고, 다른 한 명이 화면에 나온 숫자 6자리 코드를 입력하면 대전이 시작돼.
